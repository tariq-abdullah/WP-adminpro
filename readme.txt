Admin pro makes WordPress Admin back-end look more easier and beautiful. 

== Description ==
Admin pro makes WordPress Admin back-end look more easier and beautiful by putting the main menu to top of the admin backend as dropdown menu. 

Features:

* Main Menu appears as Top Bar Menu Bar 
* Progress Bar shows the status of current page load
* Admin Panel has more space and utilizes full width

== Installation ==
Upload Plugin either by uploading it directly via admin panel or to your wp-content/plugins folder and activate the plugin. No settings needed, this plugin is plug and play

== Frequently Asked Questions ==

= I don't see any change after installing this plugin ? =

This plugin is aimed at changing the look of your WordPress Dashboard when you are on a desktop with at-least 1050px screen width. Please open your dashboard in Maximized mode on a desktop or a device with atleast 1050px screen width.


= Where is the settings page for this Plugin? =

This Plugin is Plug and Play, There is no settings page for it. We Plan to add this plugin with a settings page in future.
